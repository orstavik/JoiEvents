  1. setDefault(func) adds an exclusive, lowestWins default action as we are accustomed to in the native elements.
  2. addDefault(func) adds a non-exclusive, additive default action that will run in addition to any other exclusive default action.
     This is not really implemented in the browser, but is an addition that enable developers to wrap around other elements in their context.
     Additive default actions only add when there is no exclusive default action, or when the exclusive default action chosen is UNDER the same DOM
     context as it is.
